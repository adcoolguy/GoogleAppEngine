Spring Security's AuthenticationProvider vs UserDetailsService
--------------------------------------------------------------

Use and extend AuthenticationProvide for custom authentication (e.g. external identity, lockout support etc).

Use and extend UserDetailsService for simplest authentication (e.g. just userid and password based authentication).
