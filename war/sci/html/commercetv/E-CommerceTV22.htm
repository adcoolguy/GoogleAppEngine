You need to use HTML 5 capable to view this demo page. A good HTML 5 browser is Google Chrome. Due to the Amazon web app, you also need to have/enable Adobe Flash to view the shop.

↑ This is a "Shop" component. To navigate in the shop, click on it and use Up/Down/Left/Right arrow key. Click to select the product and Escape to close/exit a pop-up.
To activate the docking mode, click on any background which is not the shop or the video. You can use the same arrow key to move the shop around on the screen. Use Page Up for minimized mode and Enter to restore the shop into the full screen/normal mode. Search feature does not work in some of the docking modes.
